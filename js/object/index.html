JavaScript之对象

定义：

javascript的基本数据类型，它有很多值可以看作属性的无序集合。属性包括名字和值，属性名是包含空字符串在内的任意字符串，属性值可以是任意javascript值或者（es5）是一个getter／setter函数。属性还包括值／可写／可枚举／可配置的特性

方法总结：

继承属性

作用之一是防止库函数无意间修改那些不受你控制的对象。因为不是将对象直接作为参数传人函数，而是将它的继承对象传人函数。所以函数操作的都是继承对象。

getter和setter

1.使用存取器属性定义API,API提供了表示同一组数据的两种方法。
2.智能检测属性的写入值以及每次读取返回不同值。

查询和设置属性API

1.可以通过这些API给原型对象添加方法，并将它们设置成不可枚举的，这让它们看起来更像内置方法。
2.可以通过这些API给对象定义不能修改或删除的属性，借此“锁定”这个对象。 3.属性描述符对象(可以通过object.getOwnPropertyDescriptor())获取某个对象特定的属性描述符 4.设置属性的特性Object.definePeoperty()
